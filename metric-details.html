How Metrics Are Calculated

Code Complexity

Measures the number of linearly independent paths through a program's source code. Lower complexity indicates more maintainable code.

Code Coverage

The percentage of your code that is executed when running tests. Higher coverage indicates more robust testing.

Technical Debt

Estimated effort required to refactor and improve code quality. Lower technical debt indicates a healthier codebase.
